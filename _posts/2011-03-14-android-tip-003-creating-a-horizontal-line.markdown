This code inserted in the a layout can be used to show an horizontal line.

```xml
<!-- your_layout_file.xml -->

<!-- ... -->
<View android:layout_width="fill_parent"
 android:layout_height="1dp"
 android:background="#FF6D7B8D" />
<!-- ... -->
}
```
